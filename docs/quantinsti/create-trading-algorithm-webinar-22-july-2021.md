## 关于会议

学习从零开始创建一个交易算法，并用真实的市场数据进行测试。了解创建交易算法的所有基本要素。本节将举例解释各种交易策略范例，如[动量交易](/momentum-trading-strategies/)和均值回归。

*   一系列策略
*   [基于动量的策略](https://quantra.quantinsti.com/course/momentum-trading-strategies) vs 基于均值回复的策略

*   基于动量算法的一个例子
*   基于均值回复算法的一个例子
*   回溯测试和前瞻测试的重要性

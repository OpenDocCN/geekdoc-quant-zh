## 关于会议

这个网上研讨会是为了让你了解算法交易的具体细节。演讲者将从 Python 的安装和与金融市场相关的库开始。

然后他将讨论制定和测试交易策略的步骤。接下来，我们还将介绍 Python 中的一个交易策略，该策略从数据下载开始，以运行回溯测试来计算一些指标进行评估为结束。

演讲者将通过暗示评估交易策略时需要考虑的其他因素来结束本次会议。

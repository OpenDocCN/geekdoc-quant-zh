**正文**

近日，微软在 iOS 发布了一款名为**微软数学**的 App：

![](img/27b0123bbc04c72386b1532bcfb6506d.png)

光看介绍，不是一般人儿：

**1、算数、初级代数、代数、微积分（求和，极限，求导数，积分）、统计（数列）通通支持。**

**2、支持打印或者手写输入。**

**3、还贴心的支持 Apple Pencil，像在纸上做题一样输入算式。**

**4、关键还有详细的解题步骤、坐标图等。**

![](img/92f1df2876e616fc560cbeededc03b84.png)

简直就是神器啊！微软出品，必须要赞！

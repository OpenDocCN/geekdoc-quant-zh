**编辑部的话**

2015 年 WorldQuant 发表了《101 Formulaic Alpha》，论文中给出了**101 个基于算法生成的 Alpha 因子**。 

![](img/ae54a8f54a05a4e866773aaa8f921bc8.png)

然后~

国泰君安在 2017 年发表了《基于短周期价量特征的多因子选股体系》的报告。该报告总共构建了**191 个基于价格和成交量数据的短周期 Alpha 因子**。

![](img/7405692f1343cc6348f7c0cb6887efa2.png) 

昨天有用户在公众号后台询问有无代码实现。**在 2017 年，我们在互联网就发现有热心网友写好了全部代码，忘记给大家分享了**![](img/b14ce9464d2c7742bdb65121e4202564.png)。今天赶紧发出来，抱歉抱歉![](img/673665aeeca09753b9c8edc414607ebe.png)![](img/673665aeeca09753b9c8edc414607ebe.png)![](img/673665aeeca09753b9c8edc414607ebe.png)

每个 Alpha 计算可能与出入，但是都有源码。大家可以进行细化、修改，然后接通自己的数据库，进行研究学习。

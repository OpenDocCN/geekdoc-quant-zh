**概述**

股票市场预测仍然是一个具有挑战性的问题，因为影响股票市场价格的因素很多，如公司新闻和业绩、行业业绩、投资者情绪、社交媒体情绪和经济因素。本研究利用深度卷积网络和 K 线图研究股票市场的可预测性。

该结果设计了一个决策支持框架，交易员可以使用该框架提供未来股价方向的建议指示。我们利用**卷积神经网络** （Convolutional Neural Network）、**残差网络**（ Residual Network ）和**视觉几何组网络**（Visual Geometry Group Network）等多种神经网络来完成这项工作。从股票市场的历史数据，我们把它转换 K 线图。最后，这些 K 线图作为训练卷积神经网络（CNN）模型的输入。这个卷积神经网络（CNN）模型将帮助我们分析 K 线图中的模式，预测未来股市的走势。该方对台湾和印尼股市数据集的预测**准确率分别为 92.2%和 92.1%**。所构建的模型已被实现为一个基于 web 的系统：*http://140.138.155.216/deepcandle/*

**方法**

模型体系结构如下。首先，使用 Yahoo!金融 API。然后利用移动窗口技术生成周期数据，再用计算机图形技术生成 K 线图。最后，将 K 线图作为输入，输入到一些深度学习神经网络模型中，寻找股票市场预测的最佳模型，输出为二分类，表示股票价格在不久的将来会上涨或下跌。

**学习算法**

我们将使用一些基于卷积神经网络的**深度学习网络**（DLN）对股票市场预测进行分类。除了 DLN 算法，我们还应用了一些**传统的机器学习算法**与 DLN 算法进行了比较。传统的机器学习算法有**随机森林算法**和**k 近邻算法**。

主要有以下：

**卷积神经网络** （Convolutional Neural Network）

**残差网络**（ Residual Network ）

**视觉几何组网络**（Visual Geometry Group Network）

**随机森林**（Random Forest）

**K 最近邻**（K-Nearest Neighbors）

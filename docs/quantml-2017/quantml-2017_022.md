今天由《新财富》杂志社主办的“第十五届新财富最佳分析师”评选颁奖盛典在深圳举行，作为专注于量化投资的我们最为关注的就是**金融工程**项目的评选，接下来，小编带你一睹为快吧！

本次入围“第十五届新财富最佳分析师”金融工程组的总共有 7 家券商的金融工程团队，分别是：**广发证券、****国泰君安证券**、**海通证券**、**华泰证券**、**申万宏源证券**、**天风证券**、**兴业证券**。
